MS Office 文件转换器

将 Office 文件转换为PDF的处理器。

运行的前提：已经安装 Microsoft Office 2010或以上版本。至少需要包含 Word Excel PowerPoint组件。

处理器完成后返回：

参数名称: file, 类型: 文件

 

Java中使用示例:

DPS dps = new DPS("http://127.0.0.1:8199/dps/server?wsdl");//使用注册服务器的地址

try {

       IProcessorManager manager = dps.getProcessorManager();

Task task = new Task();

       task.setName(name);//为你这次转换任务起一个名称，这个名称将出现在转换日志中方便查找

       task.setPriority(Task.PRIORITY_1);//默认的优先级

       task.setValue("sourceType", "docx");//docx为输入文件的类型，参考输入参数

       task.setValue("targetType", "pdf");//pdf为输出文件的类型，参考输出参数

       task.setFileValue("file", new File("D:/officetest/wordtest1.docx"));//需要转换的文件

       Result result = manager.runTask(task, "com.bizvpm.dps.processor.msoffice:msoffice.msofficeconverter");//转换器类型ID,可在DPS上查看

       result.writeToFile("file", new File("D:/officetest/wordtest1.pdf"));//转换输出的文件

} catch (Exception e) {

       e.printStackTrace();

}
